# Photos de réalisations Atom

Dépose ici les photos de tes projets.

## Format recommandé
- Format : JPG ou WebP
- Résolution : 1200×800 px minimum (ratio 3:2)
- Poids : < 500 KB par photo (utilise https://squoosh.app pour compresser)
